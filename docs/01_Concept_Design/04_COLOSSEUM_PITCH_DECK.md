## Slide 8 / 12 — Tech Stack III: Next-Level Innovation

**"Jupiter Swap. TEE Privacy. Eliza Persona."**

### Jupiter Direct Swap
- Users say *"Buy me 100 CHOCO"* — Choonsim finds the optimal route via Jupiter and executes the swap. User just signs.
- **Zero-friction, in-chat economy.**

### TEE (Trusted Execution Environment)
- Agent private keys and conversation logic run inside TEEs.
- **Not even the dev team can access user data or wallet keys.** Mathematically guaranteed privacy.

### Eliza Framework
- Consistent persona and memory across X, Discord, Telegram, and Web.
- **One soul, many platforms.** Choonsim feels alive everywhere.

### Script (3-min) — ~15s

Second, TEE — Trusted Execution Environments. We're moving our agent's keys and logic into TEEs so that not even our own team can access user data. Third, the Eliza Framework for cross-platform persona consistency. One Choonsim. One soul. Every platform."

### Script (5-min) — ~25s
> "Now let me show you what's coming next — and why it will set us apart in this hackathon. First: Jupiter Direct Swap. Right now, buying CHOCO requires leaving the app. We're integrating Jupiter's Swap API directly into Choonsim's agent toolkit. A user says 'I want to buy 100 CHOCO,' and Choonsim finds the optimal swap route, presents a confirmation, and the user signs. Done. Zero friction. The economy lives inside the conversation.
